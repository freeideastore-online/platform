# Getting Started

FreeIdeaStore is for early ideas that need public sharpening.

## Create An Idea

1. Open the console.
2. Sign in with GitHub or Google.
3. Add a title, summary, next step, and main risk.
4. Publish a short first page.
5. Invite comments, reactions, evidence, and critiques.

## First Page Target

A first idea page should usually include:

- Overview
- People And Problem
- Proposed Solution
- Risks And Constraints
- Validation
- Evolution

Raw ideas do not need to fill every section. Serious publications should grow into the universal 2-level spine documented under Publications.

## Registered Collaboration

Reading is public. Commenting, reactions, and structured contributions require a registered user so public work is attributable.
