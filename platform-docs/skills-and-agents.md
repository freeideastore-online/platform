# Skills And Agents

FreeIdeaStore publishes canonical idea skills as Markdown files and through MCP.

## Published Skills

Published skills live under `/skills/` and include playbooks for:

- Flow orchestration
- Interviewing
- Document architecture
- Critique
- Competitor finding
- Research
- Validation planning
- Prototype planning
- Refinement
- Pivot generation
- Pro-candidate assessment

## Agent Flow

1. Use Idea Flow Orchestrator.
2. Ask only missing questions.
3. Use Idea Document Architect for the first page.
4. Use the universal 2-level spine for serious canonical updates.
5. Add comments or contributions for uncertain work.
6. Publish canonical updates only with owner approval.

## Publishing Rules

- Use `##` headings for top-level publication chapters.
- Use `###` headings for sub-sections inside a chapter.
- Link named products, companies, services, datasets, regulators, and public sources when credible URLs are known.
- Do not invent evidence, users, metrics, or sources.

## Local Skills

Codex, Claude, or other clients may keep local shortcut skills, but the platform-published playbooks are canonical.
