# Contribution Prompts

- Add examples of excellent ASX company reports.
- Map competitors and newsletters already serving Australian retail investors.
- Define a valuation score that is transparent and conservative.
- Find legal guidance on financial advice boundaries in Australia.
- Add public examples of filings where valuation looked cheap but the company was a value trap.
- Propose a no-licensing MVP data workflow using user CSVs and public reports.
- Design the weekly watchlist layout with a bullish case, bearish case, and source evidence.
- Test one manual report with real Australian retail investors.
- Identify wording that keeps the product clearly inside research assistance rather than personal financial advice.
