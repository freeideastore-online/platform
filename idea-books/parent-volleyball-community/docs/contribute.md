# Contribution Prompts

- Set up the Saturday group in BAND and document what works.
- Set up the same workflow in Spond and compare friction.
- Interview 10 parents after a few weeks.
- Write the photo consent and child data rules.
- Track families joined, attendance, repeat attendance, photos shared, birthday reactions, and new families.
- Find another informal family group with the same problem.
- Identify what would make this worth building instead of just using BAND.
