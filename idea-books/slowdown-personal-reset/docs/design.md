# Design Sketch

The core flow:

1. Choose current state.
2. Take a short guided pause.
3. Write one sentence.
4. Pick one next action.
5. Review streaks and patterns lightly.

The check-in inputs should stay simple:

- mood;
- energy;
- mental speed;
- tension.

The recommendation engine should return:

- one short practice;
- why this practice fits the current state;
- a tiny next action;
- optional save or discard.

Design constraints:

- do not overwhelm with content;
- do not reward compulsive checking;
- do not frame streaks as moral success;
- make offline/local use feel complete;
- make sync optional and transparent.
