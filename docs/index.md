# FreeIdeaStore Project Docs

FreeIdeaStore hosts public idea pages and deeper project docs. The store is the browsing surface: each card shows the signal, maturity stage, risk, and a short preview. The project docs are where the real work accumulates.

## What Lives Here

- Brainstorming notes that explain why the idea exists.
- Design sketches and workflow decisions.
- Research notes, competitor questions, and evidence requests.
- Prototype plans that can be built by contributors.
- Validation steps that decide whether to support, pivot, park, or trash the idea.
- Contribution prompts so people know how to improve the project.

## How It Should Feel

The store should feel like a public idea market. The docs should feel like a working room. A visitor can skim the storefront, open a promising idea, and see enough structured thinking to decide how to help.

## Graduation Path

An idea graduates when the problem is clear, the evidence is credible, the prototype path is scoped, and contributors have produced enough judgment to justify ProIdeaStore review.
