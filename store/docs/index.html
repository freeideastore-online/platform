FreeIdeaStore Docs¶

FreeIdeaStore is a public workspace where ideas start cheap, attract useful comments and reactions, and mature through evidence, critique, validation, and prototype work.

The store page for an idea should stay easy to scan. When an idea needs more depth, it can become a multi-page publication with research, design, prototype, validation, and open-question pages.

What This Docs Site Covers¶

  • How ideas move from rough prompt to public page.
  • When an idea should remain one page.
  • When an idea deserves a multi-page publication.
  • How registered users comment, react, contribute, and refine.
  • How MCP agents use published FIS skills.
  • How FreeIdeaStore connects to ProIdeaStore.

Core Rule¶

Do not make every playful free idea into a heavy project. Start small, then add structure only when the idea earns it.
